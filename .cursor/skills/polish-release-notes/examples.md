# Note sulla versione polacca — esempi

Pattern per [SKILL.md](SKILL.md). Polacco **only** del corpo `###`, non intestazioni, non frontmatter.

## Lunghezza paragrafo

**Prima (un blocco lungo, esercitazione implicita):**

```markdown
### Example feature

You can now do X by opening [!DNL Create], going to the template gallery, selecting an approved experience, clicking **[!UICONTROL Translate]** in the actions menu, choosing one or more languages from the list, waiting for the canvas to load, and then editing the translated text before you request approval. This uses Azure Open AI and helps you scale globally.
```

**Dopo (2-3 frasi per paragrafo, con esito positivo, senza elenco passaggi):**

```markdown
### Example feature

**[!DNL Translate and localize]** now turns approved experiences into localized variants in dozens of languages—fast bulk translation powered by Azure Open AI, with room to refine copy before review.

You scale global campaigns without standing up a separate translation project for every asset. See [Manage assets and experiences](/help/user-guide/content/manage-assets.md) for where to start from [!DNL Content].
```

## Eccitazione senza pretese eccessive

Preferisci le vittorie di **concrete** (&quot;stessa dashboard&quot;, &quot;aggiornamento giornaliero&quot;, &quot;un flusso&quot;) rispetto ai vaghi aggettivi (&quot;sorprendenti&quot;, &quot;best-in-class&quot;) a meno che l&#39;origine non li supporti.
